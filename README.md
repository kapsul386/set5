## Repository structure

- **HyperLogLog.cpp**  
  Стандартная реализация алгоритма HyperLogLog.  
  Генерирует поток данных и файл `results.csv` с результатами экспериментов.

- **HyperMegaLogLogProMax.cpp**  
  Оптимизированная (FAST) версия алгоритма HyperLogLog  
  с уменьшенным потреблением памяти и ускоренным вычислением оценки.

- **plots.py**  
  Python-скрипт для построения графиков по данным из `results.csv`.

- **A3 set5.docx**  
  Отчёт по заданию с описанием реализации, экспериментов  
  и анализа точности алгоритма.

- **1.png – 5.png**  
  Графики, полученные в ходе экспериментов:
  - сравнение истинного значения и оценки;
  - средняя относительная ошибка и дисперсия;
  - детальный анализ выбранного параметра \(B\).
